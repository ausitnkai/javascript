*1* 網址 Web Addres(URL)
        不同的網址，代表網路上不同的服務或資源
        網站服務就是由各式各樣的網址組成的綜合呈現

*2* 網址的組成 ==> 通訊協定://主機名稱/路徑
        通訊協定 (Protocol)
        主機名稱 (Hostname)
        路徑 (Path)

*3* 網站前後端的互動基礎 ==> 透過網址，前端可以連線取得的資源
        1. 在瀏覽器網址列上輸入網址，按下 enter 取得網頁
        2. 可以採用 AJAX/XHR 技術執行 網址連線、取資料等動作

*4* AJAX/XHR 利用 JavaScript 程式進行網路連線
    1. 初期稱作 Asynchronous JavaScript And XML 技術
    2. 中期使用 XMLHttpRequest 物件，稱為 XHR 技術
    3. 近年，建議採用最新的 fetch 函式來執行網路連線功能
